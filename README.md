
# Employee Attrition Prediction Using Machine Learning

## Project Overview
This project aims to predict whether an employee is likely to leave the company using machine learning techniques. The dataset contains employee demographic, job-related, and satisfaction information.

## Dataset Source
IBM HR Analytics Employee Attrition Dataset from Kaggle:
https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset

## Model Used
Logistic Regression

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## Steps to Run the Project
1. Clone the repository.
2. Open `employee_attrition.ipynb` in Jupyter Notebook.
3. Run all cells sequentially.
4. View model results and visualizations.
